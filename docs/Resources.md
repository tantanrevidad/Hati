## 📖 Glossary (Stellar & Fintech Terms)

| Term | Definition |
| --- | --- |
| **Horizon API** | Stellar's REST API for interacting with the network — submitting transactions, querying accounts/balances |
| **Trustline** | An explicit opt-in on Stellar allowing an account to hold a specific non-native asset (e.g., a stablecoin) |
| **Anchor** | A regulated entity that bridges fiat currency and Stellar-network assets (on/off-ramp provider) |
| **Fee-bump transaction** | A Stellar mechanism allowing one account to pay the transaction fee on behalf of another |
| **SEP-24** | Stellar Ecosystem Proposal defining the interactive flow for anchor-hosted deposit/withdrawal |
| **XLM** | Stellar's native asset, used for network fees and reserves |
| **Stablecoin** | A digital asset pegged to a stable value (e.g., USD or PHP) to avoid crypto volatility |
| **Custodial wallet** | A wallet where a third party (Hati) holds the private keys on behalf of the user |
| **Non-custodial wallet** | A wallet where the user holds their own private keys |
